This sample show you how you can combine the ArcGIS API for JavaScript which is build on top of Dojo with jQuery.
We recommend you to do that if you fill confortable using jQuery to manage: DOM elements, events, AJAX, etc.

Combine with Bootstrap

Bootstrap-map-js is a light-weight extension for building responsive web mapping apps with the ArcGIS API for JavaScript and Bootstrap (ver 3.x). Maps will automatically resize and recenter as the screen size changes or they can be set to a fixed size. Pop-ups will stay within the screen view and are styled to match Bootstrap.

See the Pen Cargar jQuery como un módulo by Raul Jimenez Ortega (@hhkaos) on CodePen.

Bootstrap + Calcite Theme

Calcite Bootstrap is a custom theme and a custom build system for Bootstrap. The theme is based on Calcite - a geo-centric design framework created by ESRI. Calcite Bootstrap was built for developers who have experience working with Bootstrap and would like to integrate the Calcite theme and build system to their web pages and apps.

See the Pen Cargar jQuery como un módulo by Raul Jimenez Ortega (@hhkaos) on CodePen.

jQuery Mobile

jquery-mobile-map-js is a lightweight helper library for use with jQuery mobile and the ArcGIS API for JavaScript 3.x. This library isn't required for ArcGIS API for JavaScript v4.x.
